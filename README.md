#  Bulut Depolama Kullanan Fotoğraf Galeri Sistemi

Flask ve Cloudinary kullanılarak geliştirilen bulut tabanlı fotoğraf galeri sistemi.

# Özellikler

*  Kullanıcı kayıt / giriş sistemi
*  Cloudinary ile fotoğraf yükleme
*  Favoriler sistemi
*  Çöp kutusu sistemi
*  Fotoğraf geri yükleme
*  Fotoğraf arama sistemi
*  En yeni / en eski sıralama
*  Şifremi unuttum sistemi
*  Dark mode desteği
*  Profil sistemi

## Kullanılan Teknolojiler

* Python
* Flask
* SQLAlchemy
* Bootstrap 5
* Cloudinary
* SQLite
* Git & GitHub

## Proje Görselleri

Proje kullanıcıların:

* fotoğraf yükleyebilmesini,
* favorilere ekleyebilmesini,
* silinen fotoğrafları geri yükleyebilmesini
  sağlayan modern bir web uygulamasıdır.

## Kurulum

Projeyi çalıştırmak için:

```bash
pip install -r requirements.txt
python app.py
```
